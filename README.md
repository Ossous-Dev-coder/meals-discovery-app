# Meals Discovery App 🍽️

## Overview

Meals Discovery App is a desktop application built with C# and .NET that allows users to explore meals, search recipes, and view detailed cooking instructions using data from TheMealDB API.

The goal of this project is to provide a simple and fast way to discover food recipes through a clean desktop interface.

---

## Features

* Search meals by name
* Browse meals by categories
* View detailed meal information
* Display ingredients and cooking instructions
* Show meal images for better experience
* Simple and user-friendly desktop UI

---

## Technologies Used

* C# (Core language)
* .NET Framework (Desktop application)
* Windows Forms (UI development)
* TheMealDB API (External REST API for meal data)
* HTTP Client (API requests handling)
* JSON Serialization (Data parsing from API responses)
* Asynchronous Programming (async/await for non-blocking API calls)
  
---

## Architecture

The project follows a simple layered structure:

* UI Layer: Handles user interaction
* Business Layer: Handles API requests and data processing

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Ossous-Dev-coder/meals-discovery-app.git
   ```

2. Open the solution file in Visual Studio

3. Restore NuGet packages if required

4. Run the project

5. You may need to run Visual studio as Administrator
